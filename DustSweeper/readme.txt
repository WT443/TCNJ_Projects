How to run:
- open compiler to directory source code is in
- type "ghc main.hs"
- type ".\main" on windows, or "./main" on mac

How to play: 
- enter game board options (follow prompts)
- pick move type you want: check, flag or quit (follow prompts) 
- enter coordinate of rug to make your move on in "(col,row)" format
* column and row numbers are indicated on outside of board for coordinate picking
- keep playing until win/lose or quit

Issue Log:
- only a one player game
